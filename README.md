📊 Projeto BI - Correção e Qualificação de Dados com Power BI
📌 Sobre o Projeto

Este projeto tem como objetivo demonstrar técnicas de tratamento, qualificação e transformação de dados utilizando o Power BI e o Power Query.

O cenário proposto consiste na correção de qualificadores presentes em um conjunto de dados de receitas, realizando a separação e organização das informações por meio de recursos nativos do Power Query, como:

Extração Avançada de Texto
Divisão de Colunas
Promoção de Cabeçalhos
Alteração de Tipos de Dados
Renomeação de Colunas
Limpeza e Padronização de Dados
Transformações utilizando etapas aplicadas
🛠️ Ferramentas Utilizadas
Power BI Desktop
Power Query Editor
Arquivo CSV como fonte de dados
GitHub para versionamento
